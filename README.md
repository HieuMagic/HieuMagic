## Featured Projects

### [Unbiased-RecSys](https://github.com/HieuMagic/Unbiased-RecSys)
* **What it is:** A counterfactual recommender systems framework evaluating debiased estimators in PyTorch.
* **Highlight:** Implements a time-decay propensity model (TE-IPS) to adjust for exposure decay over item age, backed by automated unit tests and an SQLite experiment registry.

### [DS_Predictor](https://github.com/HieuMagic/DS_Predictor)
* **What it is:** An end-to-end used car price arbitrage and fair market value modeling pipeline.
* **Highlight:** Implemented a vehicle-centric regression model that isolates regional mispricing signals for geographic arbitrage analysis without feature leakage.
