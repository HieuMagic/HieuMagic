## Featured Projects

### [Unbiased-RecSys](https://github.com/HieuMagic/Unbiased-RecSys)
* **What it is:** A counterfactual learning recommender system framework implementing debiasing estimators in PyTorch.
* **Tech Stack:** Python, PyTorch, SQLite, Streamlit, Git Actions, unittest.
* **Highlight:** Implements time-evolving propensity estimation to correct for time-decaying exposure bias as items age. Features automated CI/CD unit testing and a local SQLite experiment logging registry for hyperparameter audit tracking.

### [DS_Predictor](https://github.com/HieuMagic/DS_Predictor)
* **What it is:** An end-to-end used car price arbitrage and fair market value modeling framework.
* **Tech Stack:** Python, ThreadPoolExecutor (web scraping Chotot API & Bonbanh), Pandas, LightGBM, CatBoost, Scikit-learn.
* **Highlight:** Implemented a vehicle-centric modeling approach that avoids data leakage and isolates regional mispricing signals (residuals) for geographic arbitrage diagnostics.

### [Laptop-Price-Prediction](https://github.com/HieuMagic/Laptop-Price-Prediction)
* **What it is:** A regression machine learning project modeling laptop prices.
* **Tech Stack:** Python, Pandas, Scikit-learn, LaTeX.
* **Highlight:** Demonstrates modular software design with clean preprocessing, data ingestion pipelines, and configuration management, backed by a detailed academic project report.
